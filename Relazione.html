Per permettere l'upload di una pagina WEB si é reso necessario l'utilizzo del protocollo FTP (File Transfer Protocol) tramite un software (nel nostro caso un FTP Client, FileZilla), che ci permette, tramite un autenticazione, di accedere allo spazio WEB che ci é stato riservato sul server di Altervista, modificandolo in base alle nostre esigenze.
